Pyrocko can be installed on every operating system where its prerequisites are available. This document describes how to install Pyrocko on Unix-like operating systems, like Linux and MacOSX.

To install Pyrocko, first try to follow the instructions in System specific installation instructions. For additional information, refer to Detailed installation instructions. If you run into problems during installation, please let us know.
